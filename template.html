Web Accessibility

Perceivable

- 콘텐츠 간의 구분 : 공백 문자로 단어가 분리된 경우는 CSS 의 글자간격 letter-spacing을 사용한다.

Robust

- 마크업 오류 방지 : 마크업 언어의 요소는 열고 닫음, 중첩 관계 및 속성 선언에 오류가 없어야 한다.
- 콘텐츠를 개발하는 과정에서 마크업 언어의 표준을 준수하고 가능한 한 문법적 오류나 모호성이 없도록 구현되어야 한다.
